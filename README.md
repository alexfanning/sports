# sports

Python tools for sports data analysis and prediction modeling, including NBA playoff outcome simulation and probability estimation.

## Overview

This repository contains Python scripts for quantitative analysis of sports data. The current focus is NBA playoff modeling — simulating series outcomes based on team performance metrics and computing win probabilities across playoff rounds.

## Contents

- `bballPlayoffs.py` — NBA playoff simulation and analysis; loads team performance data, models series win probabilities based on regular season or adjusted metrics, and simulates playoff bracket outcomes across multiple scenarios

## Requirements

- Python 3.7+
- NumPy
- Pandas
- Matplotlib
